CSS Selectors

  • Allows us to select or identify specific HTML elements
  • All use the same workflow: select element to style, create object, write CSS rule for properties
  • 3 most popular ones:
    • Type/Tag Selector
      • Selects specific "types" of HTML elements, e.g. button, HTML tags, etc...
      • The structure is 2 curly braces, followed immediately after the selector
      • Any CSS property can go inside the curly braces to style the selected elements
        • selector {
        color: #02b3e4
        property: value
        }
      The following example is a button styled below:
      < button > GO GO Gadget... < /button >
      < style > button {
      background-color: coral;
      height:150px;
      width:300px;
      color:white;
      } < /style >
    • ID
      • ID selectors selected using a # followed by the ID name
      • <p id ="content"> This section is styled using the ID selector</p>
        #content {
        font-size:30px;
        color:violet;
        }

        This section is styled using the ID selector

    • Class
      • Styles elements by selecting the class attribute
      • Class selectors selected using a period(.) followed by the Class name
      • <div class="title">This section is styled using the Class selector</div>
        .title {
        font-size:30px;
        color:tomato;
        font-style: italic;
        }
        This section is styled using the Class selector
    • Pseudo-classes
      • CSS Pseudo-classes is a keyword added to a selector that specifies speical state of selected element(s)
      • Example, :hover can be used to change the button's color when user points or hovers over it
        • button:hover {
        color: coral;
        background: white;
        }
    • Attributes Selectors
      • Attribute Selectors are a special kind of selectors that match elements based on their attributes and attribute values
      • Generic syntax consists of square brackets [] containing attribute followed by optional condition to match against value of attribute
      • Divided into 2 categories depending on the way they match attribute value
        • Presence and value attribute selectors, and
        • Substring value attribute selectors
      • These attirbute selectors try to match exact attribute value
        • [attr]This selector will select all elements with the attribute attr, whatever its value.
        • [attr=val] This selector will select all elements with the attribute attr, but only if its value is val.
        • [attr~=val] This selector will select all elements with the attribute attr, but only if val is one of a space-separated list of words contained in attr's value
    • Multiple Selectors
      • To style to all of our headings, don't have to use redundant rules, since not scalable
      • Instead, can select multiple HTML elements in same CSS rule by separating htem with commas.
        • h1,
        h2,
        h3,
        h4,
        h5,
        h6 {
        font-family: "Helvetica", "Arial", sans-serif;
        }
      • This CSS rule defines font used for all headings. Easy to change it all in 1 place.
